# Password Manager — Java OOP Capstone Project

A simple, secure, and user-friendly desktop Password Manager built with Java Swing. This project demonstrates Object-Oriented Programming (OOP), file handling, encryption concepts, and password management features in a practical application.

---

## Features

- Add new account credentials
- Search saved accounts
- Password strength checker
- Strong password generator
- Auto-lock timer
- Backup management
- CSV export support
- AES encryption/decryption simulation

---

## Technologies Used

- Java
- Java Swing
- OOP Concepts
- File Handling
- Collections Framework
- Exception Handling
